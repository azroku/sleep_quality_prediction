# sleep_quality_prediction
N-of-1 machine learning study predicting next-night sleep quality from daytime physiological and behavioural wearable metrics (HR, HRV, SpO₂, steps, calories) over 158 nights. Includes ML, deep learning, anomaly detection, and SHAP explainability. Submitted for ML &amp; Pattern Recognition / Computational Data Analytics, ETF Sarajevo 2025/26.
